# Rotate Images

**Rotates selected image files.**

**Warning: These workflows modify the original image files in place.**

## Included workflows

- **Rotate Images Left** – Rotates images 90° counter-clockwise
- **Rotate Images Right** – Rotates images 90° clockwise
- **Rotate Image 180º** – Rotates images upside down

## What they do

- Take one or more image files as input
- Apply the selected rotation
- **Overwrite the original image files**
- Preserve filenames and file formats

## How to use

1. Select one or more images in Finder
2. Right-click → **Quick Actions**
3. Choose the desired rotate workflow

## Important notes

- **No copies are created**
- File contents are modified
- File names and extensions are preserved
- Tested on macOS 26
